# 🎓 Alumni Query Assistant

## 📌 Overview
Alumni Query Assistant is a Python-based application designed to manage student details and handle queries between students and alumni efficiently.

## 🎯 Features
- Student data storage using SQLite database
- Query management system
- Simple interface for interaction

## 🛠️ Technologies Used
- Python
- HTML
- SQLite

## 🚀 How to Run Locally
1. Clone the repository
2. Run the Python file:
   python main.py
3. Open index.html in browser (if applicable)

## 📌 Note
This project runs locally and is not deployed online.
